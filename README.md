# Sales Dashboard using Power BI

## Project Overview

This project is an interactive Sales Dashboard built in Microsoft Power BI to analyze sales performance and support business decision-making. The dashboard provides insights into revenue, profit, orders, product performance, regional sales, and customer trends.

Dataset

* Global Superstore Dataset
* Format: Excel/CSV

Technologies Used

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
* Microsoft Excel
* Data Modeling
* Data Visualization

KPIs

* Total Revenue
* Total Orders
* Total Profit
* Total Quantity
* Average Order Value (AOV)
* Profit Margin %

Dashboard Features

* KPI Cards
* Revenue Trend (Line Chart)
* Sales by Region
* Sales by Category
* Monthly Revenue Analysis
* Top 10 Products
* Sales vs Profit Scatter Chart
* Interactive Slicers
* Custom Report Tooltips
* Drill-through Page
* Time Intelligence (MoM & YoY)
* Conditional Formatting

Key Insights

* Identifies top-performing regions and products.
* Tracks monthly revenue trends.
* Compares revenue and profit.
* Enables interactive filtering by Year, Region, Category, Segment, and Ship Mode.
* Provides detailed regional analysis using Drill-through.

Conclusion

This dashboard demonstrates practical Business Intelligence skills including data preparation, modeling, DAX calculations, interactive reporting, and dashboard design using Microsoft Power BI.
